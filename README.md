## التشغيل المحلي

1. افتح ملف `landing-page.html` في المتصفح
2. أو استخدم خادم محلي:
   ```bash
   npx serve .
   ```

## النشر على GitHub Pages

1. أنشئ مستودع جديد على GitHub
2. ارفع الملفات
3. فعّل GitHub Pages من الإعدادات
4. سيتم النشر تلقائياً
